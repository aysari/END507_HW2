Kodu Çalıştırma için Yapılması Gerekenler
	1) END507_HW2 dosyası .zip uzantılı indirilir.
	2) END507_HW2-main dosyasının içindeki END507_HW2 dosyası ECLIPSE'in directory'sine kaydedilir.
	3) ECLIPSE programı çalıştırılır.
	4) File -> Open Project From File System tıklanır.
	5) Package Explorer sekmesinden END507_HW2 -> src -> default altından java dosyaları açılır.
	6) Kodlar runlanır.
